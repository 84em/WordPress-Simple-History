# Add or update translations of the plugin

Translation of the plugin takes place at https://translate.wordpress.org/projects/wp-plugins/simple-history/.
When the plugin is installed WordPress downloads language packs from

So if you want to add a translation of the plugin to your language or update the existing translations, follow the guides at
https://make.wordpress.org/polyglots/handbook/translating/first-steps/ to get started.

Don't submit language files to the [Github Repo of the plugin](https://github.com/bonny/WordPress-Simple-History/).
