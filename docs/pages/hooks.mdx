# Actions and filters

With the help of hooks in WordPress you can modify how Simple History works.

You can for example change who can see the log and customize what is logged.

For more information on how to use hooks in WordPress see the documentation on wordpress.org.

- https://developer.wordpress.org/plugins/hooks/
- https://developer.wordpress.org/plugins/hooks/actions/
- https://developer.wordpress.org/plugins/hooks/filters/

Actions and filters are prefixed with `simple_history/`.
