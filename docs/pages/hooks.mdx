# Actions and filters

With the help of [hooks](https://developer.wordpress.org/plugins/hooks/) in WordPress you can modify how Simple History works.
You can for example change who can see the log and customize what is logged.

Actions and filters are prefixed with `simple_history/`.

<Callout>
	For more information on how to use hooks in WordPress see the [hooks
	documentation on
	wordpress.org](https://developer.wordpress.org/plugins/hooks/).
</Callout>
